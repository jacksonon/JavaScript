## 示例代码：详细代码看项目内文件
* MyFirstJavaScriptDemo.html 我的第一个JavaScript示例程序<!DOCTYPE html>
```JavaScript
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8"> 
<title>示例程序核心代码</title> 
<script>
function myFunction(){
	document.getElementById("demo").innerHTML="我的第一个 JavaScript 函数";
}
</script>
</head>
<body>

<h1>我的 Web 页面</h1>
<p id="demo">一个段落。</p>
<button type="button" onclick="myFunction()">点击这里</button>

</body>
</html>
```
